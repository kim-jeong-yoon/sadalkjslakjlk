4시 발표 간단 정리

1. DLP 솔루션
    1. 데이터 유출 방지 솔루션
    2. 솔루션의 주된 목적은 “내부자”에 의한 데이터를 외부로 유출 방지 하기 위함
    3. 외부와 통신되는 모든 데이터는 “유출방지 솔루션”에 의해서 수집 / 분석 / 저장 / 빅데이터 구성과 같은 4단계로 구성 
    4. 암호화된 패킷은 SSL Proxy Server를 통해 복호화 하여 수집 및 분석 가능
    5. Tap 장비를 통하여 암호화된 패킷에 대해서 AggTap으로 전송 
    6. E-Tap 장비를 통하여 복호화된 패킷에 대해서 AggTap으로 전송 
    7. AggTap에 수집된 모든 패킷은 4단계 구성을 거침 

2. 리눅스 명령어
    1. 디렉토리 이동 ( cd ) 
    2. 파일 목록 확인 ( ls ) 
    3. 파일 내용 확인 ( cat / more / head / tail )
    4. 출력물의 문자열 검색 ( grep / egrep / awk )
    5. 치환 ( sed ) 
    6. 현재 경로 확인 ( pwd )
    7. 현재 날짜 및 시간 ( date )
 
3. 패킷 분석
    1. 패킷이란 네트워크를 통해 데이터를 송수신 받을 때 사용하는 데이터 조각 
    2. 와이어샤크 와 같은 프로그램을 통해 패킷 캡쳐 및 분석 을 할 수 있음
    3. 프로토콜 별로 패킷 구조가 다르기 때문에, 분석하려는 특정 프로토콜 패킷에 대해 구조 파악 필요함 
    4. 간단한 메일 전송 실습을 통한 패킷 캡쳐 및 분석 발표
